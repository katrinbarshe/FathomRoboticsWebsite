Mobile Emergency Stop Protection for Robotics & Industrial Safety
The Fathom Robotics Wireless E-Stop System provides a robust, fail-safe solution for immediate equipment shutdown in environments where hardwired safety systems are impractical or restrictive. Designed for robotics, autonomous platforms, and industrial machinery, this system ensures that operators maintain full safety control from a distance.
The system consists of a handheld Transmitter (TX) and a stationary Receiver (RX) that maintain a continuous, two-way heartbeat link. In the event of an emergency, a button press instantly triggers the receiver to open safety relays, halting machine operation. Crucially, the system features active fail-safe monitoring: if the wireless link is lost due to interference, range issues, or power failure, the receiver automatically defaults to a safe "STOP" state.
With a high-reliability LoRa radio link and customizable safety logic, the Fathom Robotics E-Stop is an essential safety layer for dynamic industrial environments.

Key Highlights
Fail-Safe Heartbeat Monitoring:
The receiver continuously monitors the transmitter's signal. If the connection is lost for more than 1 second, an alarm triggers; if lost for 8 seconds, the system automatically engages the E-Stop to prevent unprotected operation.
Long-Range LoRa Communication:
Utilizes 433MHz E32 LoRa radio modules for superior range and penetration compared to standard 2.4GHz Wi-Fi solutions.
Packet Integrity Check:
To prevent false restarts, the system requires 20 consecutive valid "RUN" packets before releasing safety relays, ensuring a stable connection before machinery engages.
Bypass & Maintenance Mode:
Features a dedicated hardware override button on the receiver to allow controlled machine movement or maintenance without the transmitter, complete with visual and audible safety alerts.
