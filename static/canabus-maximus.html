CANaBUS
Maximus
All-In-One Marine Autonomy Computer for Unmanned Surface Vessels
Enterprise-Grade Integrated Autonomy Platform
The CANaBUS Maximus represents the pinnacle of marine autonomy integration—a single-board computer that consolidates an entire USV autonomy stack into one rugged, production-ready package. Where previous solutions required separate devices for computation (Raspberry Pi), navigation (PixHawk Cube), and positioning (GPS/RTK module), the CANaBUS Maximus integrates all critical subsystems onto a single PCB within an IP67-rated automotive enclosure.
Built around the powerful Raspberry Pi Compute Module 5, paired with a PixHawk Cube Orange autopilot and precision RTK-capable GPS, this platform eliminates the complexity of multi-device integration while delivering professional-grade performance for commercial USV operations. The result is reduced cabling, simplified installation, improved reliability, and dramatically lower deployment time.
The CANaBUS Maximus is designed for operators who demand turnkey autonomy solutions without sacrificing flexibility. Whether running ROS2 navigation stacks, ArduPilot mission planning, or custom Fathom Companion interfaces, this integrated platform provides the computational horsepower, sensor fusion capabilities, and real-time control necessary for demanding maritime autonomy applications.
Key Highlights
Integrated Compute Module 5 Platform:
Raspberry Pi Compute Module 5 provides quad-core ARM Cortex-A76 processing at 2.4GHz with up to 8GB RAM—delivering desktop-class performance for ROS2 navigation, machine learning inference, computer vision processing, and multi-sensor fusion. Native PCIe, dual HDMI, and USB 3.0 support enable direct connection to high-bandwidth peripherals.
Embedded PixHawk Cube Orange Autopilot:
Full ArduPilot/PX4 autopilot integration with triple-redundant IMUs, barometric altimeter, and magnetometer. The Cube Orange provides L3-certified flight controller capabilities adapted for marine autonomous navigation—enabling waypoint following, geofencing, fail-safe return-to-home, and mission scripting.
RTK-Enabled Precision GPS:
AS-RTK2B module with u-blox F9P dual-frequency (L1/L2) GNSS receiver delivers centimeter-level positioning accuracy when configured with RTK base station corrections. Supports GPS, GLONASS, Galileo, and BeiDou constellations for maximum availability and redundancy.
Triple Independent CAN Bus Interfaces:
Three galvanically isolated CAN 2.0B ports (using TCAN4550 transceivers) provide simultaneous connectivity to engines, steering actuators, navigation equipment, and auxiliary systems—identical capability to the CANaBUS Mini but integrated into a unified platform.
Onboard Ethernet Switch:
KSZ8863 dual-port Gigabit Ethernet switch enables daisy-chaining of IP-based sensors, cameras, and telemetry equipment directly through the CANaBUS Maximus enclosure—reducing harness complexity and simplifying network architecture.
Expandable NVMe Storage:
M.2 NVMe slot supports high-speed solid-state storage for mission logging, video recording, map data, and neural network model storage. PCIe Gen 2 interface delivers up to 2GB/s throughput—critical for data-intensive autonomy applications.
Real-Time Clock with Battery Backup:
CR2032-backed RTC maintains accurate timekeeping across power cycles—essential for timestamping sensor data, GPS time synchronization, and mission replay analysis.
Single-Board Integration:
Replaces three separate devices (Raspberry Pi + PixHawk Cube + GPS module) with one integrated board, reducing points of failure, eliminating inter-device communication latency, and simplifying installation, maintenance, and troubleshooting.
